Context

Over the past years, social media has had an increasing influence over American politics and elections. Social media allows candidates to bypass the traditional gatekeepers of mainstream media companies by talking directly to users. Attention-grabbing tactics, heated rhetoric, breaking scandals, etc. have made millions of impressions on users on social media - as the 2020 presidential elections approaches, there is no longer any doubt in the minds of candidates and campaign staffers of the necessity of establishing a strong and impactful presence online.

At A Glance

The first graph compares engagement values (number of retweets or favorites) per tweet across multiple candidates.

Notes:

  • These are interactive graphs! Feel free to filter the graph data by using the filter controls on the left panels. One of the main reasons for developing the app this way to to provide the user with as much control as possible over the data visualization. Try out filtering the data based on relevant dates provided in this Wikipedia event timeline of the 2020 elections.
  • On initialization, data is displayed using tweets from a month prior to the current date. This general rule is true for all graphs/lists displayed on initialization.
  • Tweet publish time data is based on UTC. Therefore the app uses a UTC time basis for filtering tweets. Again, this rule is generally applied for all graphs on initialization.

Moving Averages

The second graph attempts to visualize change in data across time using moving averages. Similarly to our first viz, this graph tries to give a sense of the differences in engagement across different candidates.

Notes:

  • The "moving average" metric is arrived at by taking each date, collecting all tweets for that candidate up to nine days prior (inclusive), and averaging the data. Thus the consistency for eadh data point lies in the time range, not the number of tweets.
