A Classification Model

This model has been trained on ~40,000 tweets from candidates taking into account features such as number of retweets, number of favorites, month, day, and hour published as well as text information, with the goal of being able to predict which user published a given tweet based on this information. With 28 candidates running, a random selection has a ~3.6% chance of guessing correctly. This model has an accuracy rate of ~55% for predicting the user correctly, with plenty of room for improving accuracy by tweaking parameters. Though not perfect, this substantial increase over "pure luck" is a great indicator of how powerful even simple deep learning algorithms can be.

Notes:

  • There is strong evidence that this model was "overfit" on the training data. It achieved almost perfect results on the training data, and the drop was significant on the testing data. This was true even for as low as 5 epochs of training. There are known methods for preventing this effect.
  • A very simplistic NLP method was used for processing text data - however, the text data contains the key information for distinguishing candidates who otherwise have very similar metrics.
  • These kinds of classification models can prove to be highly beneficial for campaigns which aim to distinguish their messaging, which often falls into the trap of using bland or predictable talking points. For instance, the model will provide more or less certainty of whether different tweets belong to a candidate, even it the predicitons are correct. The "certainty" of a model's predictions can provide insight into whether the messaging of a tweet is distinct enough such that it clearly conveys a candidates' unique "brand". This is further helped by techniques which help identify which features of a tweet were most helpful in producing the model's predictions (LIME). These are typically unknown purely from the model itself. As a final thought, to the point of "distinct" or "predictable" branding, while this model has much room for improvement, one can see patterns in the model's predictions i.e. which candidates it incorrectly predicts, the levels of certainty amongst different candidates, etc.
