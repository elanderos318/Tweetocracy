A Classification Model

This model has been trained on ~18,000 tweets from candidates taking into account features such as number of retweets, number of favorites, month, day, and hour published as well as text information, with the goal of being able to predict which user published a given tweet based on this information. From a selection of the top 5 candidates, a random selection has a ~20% chance of guessing correctly. This model has an accuracy rate of ~78.8% for predicting the user correctly. By "hypertuning" paramaters within the model, we can make very accurate classification models.

Notes:

  • The top five candidates were selected for two reasons: it made the model file size small, and having 5 candidates rather than 28 allows for much greater accuracy. With previous models which were developed with all 28 candidates, the accuracy rate was about ~55% compared to a random toss of ~3.5%.
  • Text data contains key information for distinguishing candidates who otherwise have very similar metrics. On previous models which included all candidates, the model would often confuse many of the same candidates with each other (usually those with low engagement numbers).
